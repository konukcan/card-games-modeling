\section{Overview}

This report shows all pre-recorded tasks generated for reproducible experiments.
Each task has:
\begin{itemize}
    \item \textbf{Training examples}: What the model sees during training (balanced 50\% positive, 50\% negative)
    \item \textbf{Holdout examples}: Used for verification (disjoint from training)
    \item \textbf{Near-miss negatives}: Negatives differ from positives by only ONE card
\end{itemize}

\textbf{Key guarantees:}
\begin{enumerate}
    \item All tasks have equal positives and negatives
    \item Seed positives (used to generate near-miss negatives) are SEPARATE from training positives
    \item Holdout is completely disjoint from training
\end{enumerate}
